# 명리당(命理堂) · 사주 소개 사이트

전통 한국 스타일(수묵화·한지·단청 색)의 사주 소개 정적 웹사이트입니다.
사주를 보고 싶은 분이 **카카오톡 채팅으로 상담을 신청**하도록 안내합니다.

## 구성
- `index.html` — 메인 (히어로, 사주 소개, 서비스 카드, 카톡 상담)
- `services/` — 종합사주 · 궁합사주 · 자녀사주 · 오늘의 운세 소개
- `css/style.css` — 스타일
- `js/main.js` — 캐러셀, 메뉴, 카톡 링크 주입

## 카카오톡 링크 설정
`js/main.js` 상단의 한 줄만 바꾸면 사이트 전체 카톡 버튼에 적용됩니다.

```js
const KAKAO_LINK = "https://open.kakao.com/o/본인링크";
```

## 미리보기
`index.html` 을 브라우저로 열면 됩니다. (정적 사이트라 별도 빌드 불필요)
